# NotePad_App
NotePad App to store your notes n to do list using SQLite

##  Features
<ul>
<li>To DO List</li>
<li>Swipe Right/Left to Delete</li>
<li>Long Press to Edit Note</li>
<li>Adaptive Layout</li>
</ul>
